Create Virtual Environment

Set up an isolated Python environment

python3 -m venv venv

3

Install Dependencies

Install all required Python packages

pip install -r requirements.txt

Configure your initial settings

python3 setup.py
6

Start GhostOffice

Launch the dashboard

python3 main.py

❓ FAQ

Does it work on Windows?

Yes! You can run GhostOffice natively with Python, or use Docker for a containerized setup. WSL2 also works great.

Which AI model works best?

We recommend phi3:mini — it's fast, lightweight, and runs well on modest hardware without sacrificing quality.

Is my email password stored?

Never! Your password is used once to create an AES-256 encryption key via PBKDF2 (600K iterations). The key is derived at runtime and never written to disk.

Does it need internet?

No. GhostOffice is 100% offline. All AI processing happens locally via Ollama. Emails are downloaded via IMAP but never sent anywhere else.
